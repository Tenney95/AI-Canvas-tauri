# 2026-06-19

- 修复 React Flow 节点内所有 popup/dropdown 点击外部不关闭问题：`QualityRatioSelector`、`VideoParamSelector`、`ModelSelector`、`MentionEditor`、`SlashCommandMenu` 的 `document.addEventListener('mousedown', handler)` 统一改为捕获阶段 `addEventListener('mousedown', handler, true)`。根因是 React Flow 画布的 pane 层会拦截/停止冒泡阶段的 mousedown 事件，导致 document 冒泡监听器收不到事件。捕获阶段在事件到达目标前先触发，不受 React Flow 拦截影响。Sidebar/NodeMenu/AssetsPanel 等顶层组件不在 React Flow 内，无需修改。
