## App.tsx 毛玻璃效果优化

- 将根容器背景改为 `bg-canvas-bg/80` + `backdrop-blur-2xl`
- 第一次尝试：Rust 端 `window.set_effects(WindowEffectsConfig { effects: [Effect::Acrylic] })` 启用 Windows 原生 Acrylic — 效果覆盖整个窗口，左侧透明区和圆角都被渲染，不够精确
- 最终方案：回退 `lib.rs` 的 `set_effects`，仅用 CSS `backdrop-blur-2xl` 在内部 content div（`rounded-[16px] overflow-hidden`）上，模糊被精确限制在圆角内容区域内
- 注意：CSS `backdrop-blur` 能否在 WebView2 透明窗口下模糊桌面取决于渲染管线实现，若不生效需另寻方案
