## 新增：本地图片上传到远端图床

**问题**：`aiService.ts` 中 `resolvePromptToChatContent()` 和 `resolvePromptWithImageRefs()` 直接将本地 `asset://localhost/...` 或 `data:...` 图片 URL 塞进发给远程 AI API 的请求体，远程 API 无法访问这些本地资源。

**改动**：

1. **新文件 `src/services/uploadService.ts`**：
   - `isLocalImageUrl(url)`: 判断是否为本地 URL（data: / asset:// / file://）
   - `uploadToRemote(url)`: 上传单张本地图片到远端图床，非本地 URL 直接返回；带内存缓存避免重复上传
   - `uploadAllToRemote(urls)`: 批量上传

2. **修改 `src/services/aiService.ts`**：
   - 新增 `resolveContentImageUrls(content)`: 处理 `generateText()` 中多模态 content 数组里的本地图片 URL
   - 新增 `resolveImageUrlArray(urls)`: 处理 `generateImage()` 中 `image_urls` 数组里的本地图片 URL
   - `generateText()`: 调用 `resolveContentImageUrls()` 后再发送
   - `generateImage()`: 调用 `resolveImageUrlArray()` 后再发送（包括 APIMart 异步路径）

**未改动**：`apimartService.ts` 角度控制流有自己的 `uploadToApimart()` 处理 data URL，asset URL 场景是已有问题，不在本次范围。

**数据流**：本地图片 → `uploadToRemote()` 用 APIMart `/uploads/images` 上传 → 公网 URL → 替换原 URL 发送给 AI API

## MiniMap 体验优化
- 参考 @xyflow/react 官方文档，对 `Canvas.tsx` 的 `<MiniMap>` 做了 6 项改进：
  1. **`pannable` + `zoomable`** — 用户可在小地图上拖拽平移/滚轮缩放视口
  2. **`nodeColor` 按类型着色** — ai-text=indigo, ai-image=green, ai-video=blue, ai-audio=orange, group=neutral
  3. **`nodeStrokeColor` / `nodeStrokeWidth`** — 添加节点描边提高可辨识度
  4. **`nodeBorderRadius`** — 节点圆角 = 4，匹配设计
  5. **`maskStrokeColor` / `maskStrokeWidth`** — 视口框线 indigo 色，更醒目
  6. **`bgColor` prop** — 替代 style.backgroundColor 内联样式

- 修复：CSS className `!w-[180px] !h-[120px]` 导致遮罩偏移 — MiniMap 组件从 `style.width/height` 读取尺寸计算 viewBox 和蒙版，CSS 宽高它感知不到 → 将宽高移入 `style` prop，className 只保留定位偏移

## MiniMap 遮罩/视口框圆角修复

基于 xyflow MiniMap 源码分析：
- 遮罩是 SVG `<path class="react-flow__minimap-mask">` 用 `fillRule="evenodd"` 画外框+视口内框两个矩形
- 视口框线（紫色）是内框矩形的 stroke，通过 CSS 自定义属性 `--xy-minimap-mask-stroke-*` 注入
- SVG `<path>` 的 fill 不能用 CSS `border-radius`，但 `stroke-linejoin: round` 可圆化描边拐角

**修改**：
- `reactflow.css`: 新增 `.react-flow__minimap { border-radius + overflow + box-shadow }` + `.react-flow__minimap-mask { stroke-linejoin: round !important }`
- `Canvas.tsx`: `maskStrokeWidth` 1→3（stroke 太细看不出圆角），`borderRadius` 8→10px（与 CSS 统一）
