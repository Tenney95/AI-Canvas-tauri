## 2026-06-08

### GooeyBtn 颜色跟随 Handle 类型色
- GooeyBtn 新增 `hue` prop（number），默认 170，通过 inline style `--hue` 覆盖内置 hue
- TextNode 传入 `hue={234}` 使 GooeyBtn 显示 indigo 色，匹配 `.handle-text` 的 `#6366f1`
- 颜色映射：text=234°（indigo）, image=142°（green）, video=217°（blue）, audio=30°（orange）

### ImageNode / VideoNode / AudioNode 添加 GooeyBtn
- 三个节点的 Handle 中均添加 `GooeyBtn`，颜色跟随各自 Handle 类型
- ImageNode: `hue={142}`（green .handle-image #22c55e）
- VideoNode: `hue={217}`（blue .handle-video #3b82f6）
- AudioNode: `hue={30}`（orange .handle-audio #f97316）
