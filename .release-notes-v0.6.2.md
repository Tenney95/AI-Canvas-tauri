## ✨ 版本摘要

AI Canvas v0.6.2 聚焦图片编辑体验升级（小洛摄影棚）、Agent 配置与联网搜索完善，以及 Provider 与模型配置的稳定性修复。

## 🎨 图片与摄影棚

- 接入小洛摄影棚，并移除旧角度控制。
- 移除摄影棚面板圆角边框，统一视觉规范。

## 🤖 Agent 与搜索体验

- 完善联网搜索、会话持久化与配置体验。
- 保存厂商配置后自动打开 API Key 设置补填密钥。
- 修复模式降级、项目重验、消息序号与项目删除并发问题。

## 🔧 Provider 与模型配置

- 增强 Gemini generateContent 协议导入与文档站过滤。
- 暴露真实读取错误并放宽解析与大小限制。
- 收敛模型调用配置与请求传输。
- 删除服务商时清理所有相关模型引用。

## ✂️ 架构与样式

- 清理孤岛组件与遗留样式，降低维护成本。
- 移除摄影棚面板圆角边框，风格更统一。
