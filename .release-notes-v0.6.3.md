## 版本摘要

AI Canvas v0.6.3 聚焦图片生成兼容性、生成历史与画布操作体验，并进一步提升素材管理、模型调用和对话 Agent 的稳定性。

## 图片生成与参考图

- 图片节点新增生成历史，可按节点查看历次成功结果并全屏预览，本地文件不可用时自动回退到远程结果。
- 自定义图片模型新增参考图请求方式：默认兼容 `/images/generations` JSON `image_urls`，也可选择 `/images/edits` Multipart `image[]` 文件上传。
- 原生 Tauri 请求传输支持 FormData 序列化和 multipart boundary，单张、多张参考图均可正确发送。
- 图片接口返回 HTML 或非 JSON 内容时提供明确诊断，便于发现连接地址缺少 `/v1` 等配置问题。
- 生成素材会从提示词中提取并持久化标签，方便后续整理和检索。

## 画布与素材体验

- 节点拖拽新增等间距吸附，并优化图片生成期间的画布交互性能。
- 修复裁切比例越界与缩放交互，提升图片编辑的稳定性。
- 素材瀑布流新增列数设置，并修复开关、下拉菜单和主题样式适配。
- 修复浮层滚动条定位偏移，为弹窗补充窗口拖拽区域。

## 模型与 Agent 稳定性

- 收敛媒体 Provider 注册与 APIMart 适配，统一图片、视频和音频的模型调用边界。
- 拆分对话执行、单轮 Agent 执行与独立窗口同步控制器，降低长会话和多窗口状态串扰。
- 修复项目切换、对话消息序号和异步持久化的一致性问题。
- IndexedDB schema 升级至 v14，完善项目生成设置和历史数据持久化。
